# AI Engineer Portfolio

A modern, minimal, and high-performance portfolio website built for AI Engineers.

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Language:** TypeScript

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

3. **Build for production:**
   ```bash
   npm run build
   ```

## Deployment (Vercel)

The easiest way to deploy your Next.js app is to use the [Vercel Platform](https://vercel.com/new).

1. Push your code to a GitHub repository.
2. Log in to Vercel and import the project.
3. Vercel will automatically detect the settings.
4. Click **Deploy**.

## Customization

- **Projects:** Edit `lib/constants.ts` to update your projects and skills.
- **Components:** Reusable UI components are located in `components/ui`.
- **Colors:** Update `app/globals.css` or `tailwind.config.ts` to change the theme.

## Features

- **Responsive Design:** Works on all devices.
- **Dark Mode Ready:** CSS variables set up for easy theming.
- **SEO Optimized:** Meta tags and OpenGraph setup in `app/layout.tsx`.
- **Performance:** Optimized with Next.js best practices.
